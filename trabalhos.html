1. Página de Login — “Login Simples e Funcional” Uma interface de autenticação moderna, construída com HTML, CSS e JavaScript. Apresenta campos de entrada para nome de usuário e senha com verificação básica e responsiva, ideal para aplicações web iniciais. Foco na simplicidade visual e na experiência do utilizador.



2. Galeria de Fotos — “Galeria Interativa” Um layout limpo e responsivo para exibição de imagens. Utiliza grade CSS para organizar as fotos com efeito hover e ampliação ao clique. Perfeita para mostrar portfólios visuais ou momentos capturados. Desenvolvida com foco em leveza e usabilidade.



3. Landing Page — “Mukanda Web” Uma página profissional que apresenta os serviços da Mukanda Web, uma startup de desenvolvimento de sites. Visual forte com tons escuros e detalhes em azul claro, transmite identidade africana e tecnologia. Inclui seções sobre, serviços, depoimentos e chamada à ação.



4. Site de Vendas — “Mukanda Shopping” Um e-commerce funcional com catálogo de produtos, categorias e layout moderno. Desenvolvido em HTML, CSS e JavaScript, simula a experiência de uma loja virtual, com foco em praticidade, clareza de informação e visual atrativo.
